CDS269

Copy DD01 Data Dictionary Records

 Last Revised: 08/20/04

This program is used to copy one or more XXDD01 data dictionary records from one company to another.  Note that the DD01 data dictionary files are no longer used, but this program currently remains available for backward compatibility.  Refer to CDS181 to copy the current Data Dictionary Records.

Calling Format:

CALL "CDS269",ERR=XXXX,Y$,FROMCO$,TOCO$,FILENAME$

CDS269
XXXX is the line to branch to if:
the data dictionary file does not exist for FROMCO$ or TOCO$ or
if FROMCO$ and TOCO$ are the same.
Y$ is the terminal control variable
FROMCO$ is the company code where the copy is from. 
You may specify XX for the current company in Y$.
TOCO$ is the company code where the copy is to. 
You may specify XX for the current company in Y$.
FILENAME$ is the starting characters of the files to be copied.  All matching records will be copied.

Note: If no matching records exist, no error will be generated. 
Existing data dictionary records are overwritten.

Examples

CALL "CDS269",Y$,"DY","XX","AP10      "

will copy data dictionary for file AP10 from company DY to the current company.

CALL "CDS269",Y$,"DY","XX","AP"

will copy all data dictionary entries from company DY to the current company that begin with AP.
