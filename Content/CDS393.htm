CDS393

Obtain list of processes that have a file open

 Last Revised: 01/29/09

This routine is called by the error routine (CDS063) and the file open routine (CDS095) to obtain a list of process ID's that have a file open.  This list is used in conjunction with a call to CDS037 to display session information for the processes in the list.

CALL "CDS393", Y$, FILNAM$, S393$

CDS393
Y$ Standard Task Control Variable
FILNAM$ The name of the file.  Can use either the base name or full path name.  Dataserver prefix is supported.
S393$ A string of 4 byte process ID's in BIN(PID,4) format.

S393$ will be returned as null on Windows systems, and when the Command Processor is not running, and if FILNAM$ is passed as null.  The command processor is not required on AIX systems.
