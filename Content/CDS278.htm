CDS278

Copy Menus

 Last Revised: 10/25/01

This called routine can be used to copy menus from one menu file to another.

It will replace menu's that already exist, and add menu's that do not. If the output menu file has a fixed maximum number of records or does not have 2560 characters per record, this routine will convert the file to have unlimited number of records, and change the characters per record to 2560.

Prior to the menu copy, this routine saves menu security levels, and restores them after the copy.

CALL "CDS278",FRSI50,TOSI50,MENUS$

CDS278
FRSI50 the channel number of the copy from menu file, as opened by the calling program
TOSI50 the channel number of the copy to menu file, as opened by the calling program
MENUS$ a string of three character menu codes to be copied
