CDS604

Return Template for Form

 Last Revised: 03/14/11

This routine is used to return an empty templated data structure used with Dynamo Form Processing to provide all variable information to place on the form.

After retrieving the templated data structure, set the values to be printed on the form, then call CDS606 to merge the variable data with the form, and send the completed form to a printer or PDF file.

When using the Dynamo Tools Program Editor, CDS604 will also set the global used to enable code completion for fields contained on the form, similar to other data structures defined in the Data Dictionary.

Calling Format:

CALL "CDS604", ERR=XXXX, Y$, FORMID$, REC$

CDS604 Arguments
Argument Passed To/From Description
ERR=XXXX N/A Error branch taken if:
FORMID$ was not found in the Form File as defined using Form Processing (FMS).
Unable to dimension the template.
Y$ To Session Control Variable
FORMID$ To The FormID as defined using Form Processing (FMS).
REC$ From The dimensioned data structure used to set all form field values to be printed on the form.
Example
CALL "CDS604",Y$,"FSM1",FSM1$

CALL "CDS279",FSM1$
