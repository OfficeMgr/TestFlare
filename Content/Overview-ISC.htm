Inter-System Communication

These programs are used by developers to access a remote system and to transmit and receive files and programs and to replicate transactional data between servers.

Note that these applications have been deprecated and are no longer recommended for use. The basis data server, bbj replication services, rsync, ftp and sftp are better options.

They can be run from the menu, and be run as a background task following steps defined in a queue.  The connection to the remote system can be via direct connection to serial port, modem, local network, or internet.  These programs are only available on Unix   An operating system typically run on servers for multiple users that use character based terminals, networked personal computers and terminals. On IBM Risc System 6000 computers, IBM calls their Unix operating system AIX. On Intel based systems such as the IBM Netfinity series, SCO OpenServer 5 is one of the popular Unix operating systems. based systems.

These programs, when used in conjunction with the CDS036 called routine, can be used to synchronize files between multiple systems without having to replace the entire file.
