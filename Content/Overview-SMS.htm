runtime components

Overview

 Last Revised: 06/05/14

This software includes:

Routines called by application programs including:
  • A comprehensive data entry and validation routine used for all operator entries including numeric, string, date and window text
  • an error routine to process all program errors and interruptions
  • A routine that is used to open all data files and return channel, template and key definition data
  • A printer selection routine including faxing and emailing
  • Template management routines
Application Menu Processor including:
  • File based menus
  • Login and task security with optional log
  • Hotkey capability to switch to another application and return to original
  • Comprehensive system and task information display
  • Disc usage display
  • File usage and expansion
  • Installed software module and version tracking
  • Keystroke memory recording and playback
  • Multi-company
  • Files opened and locked by menu, including assignment of channel numbers, retrieval of string template, and key generation variable
  • An integrated internal messaging system
  • Memo writing/printing/faxing/emailing
  • Display of running jobs with Unix kill option
  • Direct interface to DoubleVision remote control application
Unix scripts including:
  • Profiles for user , root and inter-system communication logins
  • System backup and tape verification
  • Termcap/terminfo files for scoansi, Wyse 60/160 and IBM3151 terminals, Facetwin and PowerTerm terminal emulators
  • Script to run BBx processes in the background
  • Text to append to /etc/magic file for Unix file verb
  • Unix script for SCO OpenServer 5 operating system spooling
Programs attached to menus:
  • System Management Software Menu (SMS)
  • Printer & Fax Management (PFM)
  • System Management Configuration (SMC)
  • System Management Utility Functions (SMU)
