CDS059

Change CTL or ERR Value

 Last Revised: 06/09/2009

This program can be called to set the CTL system variable to any value from 0 to 255, or the ERR system variable.

This may be desired when a program tests for a particular CTL value, but is not set naturally by the program.

Calling Format:

CALL "CDS059",ERR=NNNN, CTLERR

Argument Description
ERR=NNNN Branch to this line when setting ERR
CTLERR Zero or positive integer: Set CTL
Negative integer: Set ERR to -CTLERR
Example: Set CTL=4
?CTL

 0

CALL "CDS059",4; ?CTL
 4

Example: Set CTL=200
CALL "CDS059",200; ?CTL

 200

Example: Set ERR=99
0100 CALL "CDS059",ERR=0110,-99
0110 PRINT ERR

 99
