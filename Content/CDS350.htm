CDS350

DOS Diskette Functions

 Last Revised: 11/05/03

This program can:

  1. Copy files from DOS diskette to hard drive
  2. Copy files from hard drive to DOS diskette
  3. Format a DOS diskette

It may be run (from the SMU menu) or called.

Calling Format:

CALL "CDS350",ERR=NOGO,Y$,PAR$,FILIN$,FILOUT$

CDS350
NOGO the line number to branch to if unable to perform the requested operation
Y$ the standard terminal control variable

PAR$(1,1)

the action code

1 copy from diskette
2 copy to diskette
3 format diskette
blank prompt operator for action
FILIN$ the input file name, if blank the operator may select from the files in the PRT/ directory
FILOUT$ the output file name, if blank, then the output file will be the same as the input file
Notes:

  • This program is only available on AIX systems at this time. If you could use this program on SCO Unix , DOS, or Windows systems please notify Len Krause.

  • Copying from diskette: All output files are stored in the "import" directory unless a file name is passed to CDS350.

  • If output file exists, the operator will be warned prior to copy.

  • Copying to diskette: Input file must be selected from PRT/ directory unless file name is passed to CDS350.

  • Output files that exist are replaced without warning.

  • All output files must be in the root diskette directory.

  • Input file cannot exceed formatted diskette capacity 1.44Mb.

  • Input file cannot be zero characters.

  • Input file cannot be a directory.

  • Wild cards are not supported.

Formatting:

  • Only 1.44Mb diskettes are supported.

  • AIX does not have a way to suppress prompts and output.

  • Operator must answer N to format another.
