CDS060

Text Printing

 Last Revised: 02/19/05

This program is not normally called by application programs. It is used by the Text Printing and Documentation Manual Printing programs to print text created with the Text Editor program to a printer.

Calling Format:

CALL "CDS060", ERR=1234, Y$,G, T2$, G1, L$, L, P, C$, T$, C0$, K$, K0$, F, T3$

CDS060
Y$ the standard terminal control variable
G indicates characters to be indented. Text is printed @(G)
T2$
(5,1) Print text code
Y - yes
N - no
S - selector summary text only
(6,1) print missing text list (Y/N)
(7,1) continuous paper/single fed sheets (C/S)
(8,1) print text id at top of each page (Y/N)
(9,2) company code for text opened
(13,1) lower case printer (Y/N)
G1 number of extra blank lines to insert
L$ no longer used, pass as null
C$ documentation control number
T$ title to use
C0$ company name to use (no page headings if null)
K$ key to text file including company code, but excluding page number (8 characters) 
K0$ local use only, pass as null
F channel number to use as text file
T3$ (1,1) update with new documentation control number (Y/N)
(2,1) force new title (Y/N)
