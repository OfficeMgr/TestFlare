CDS093

Obtain Program Statement

 Last Revised: 10/10/13

This routine is called by certain Dynamo Tools programs such as the Program Utility, to obtain a program statement in listed format. It would not normally be called by an application program.

Calling Format:

1010 U5=-1
1020 CALL "CDS093", ERR=XXXX, Y$, CHAN, PROG$, BUF$, VBTBL$, STMT$, ERRNO, LINEREF$, LINETYP, CPLSTMT$, CVS

. . .

10XX GOTO 1020

CDS093
Y$ standard terminal control variable
CHAN Channel number to be used by CDS093 for accessing program
PROG$ name of program to access
BUF$ used as a disc buffer variable by CDS093. You must pass a variable name, not "", which you do not use in the calling program.
VBTBL$ used to hold variable tables (if any). Like BUF$, you must pass a variable name, not "", which you do not use in the calling program.
STMT$ program statement, or "END"
ERRNO set to -1_by calling program before calling the first time. Set by CDS093 to 0 or to the error number if the program statement has an imbedded error.
LINEREF$ Set to "Y" by CDS093, when running bbx, for bbx programs only, program statement may contain a reference to a line number or label
LINETYP Set when running bbx, for bbx programs only
0Other statement
1REM or Line Label
2TABLE, IOLIST, DEF, SETERR, SETESC
CPLSTMT$ Set to compiled program statement when running bbx, for bbx programs only
CVS May be passed to CDS093 to adjust STMT$ using CVS(STMT$,CVS). For example, pass as 4 to return STMT$ in all upper case letters.
