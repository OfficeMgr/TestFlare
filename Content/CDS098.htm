CDS098

Hotkey / Help Interface

 Last Revised: 04/26/10

This called routine is used to perform help text, hotkey, and screen print processing. It is called by the input routine CDS069 and other programs where entries are made. It can also be called in console mode (with no arguments) to invoke the hotkey menu.  The hotkey menu is normally invoked by touching F1 at any entry.  The selections on the hotkey menu are specified by company using Menu Control Maintenance (SMC).

The hotkey menu automatically includes the 'Other Menu Selection' option, which is assigned Hotkey 'Y', and the 'Print Screen' option, which is assigned Hotkey 'Z'.  If these hotkeys are already assigned to another application, then a lower case hotkey letter will be assigned to these functions.

If the selected application is a 'Run' type menu selection, then a new bbx instance is used so that there are no conflicts with open file channels, variables in memory, etc.

Applications started by Hotkey will have the hotkey level shown on 'Display System Status' (SMS).

Example:
CALL "CDS098"
