CDS185

Enter Directory/Folder (T)

 Last Revised: 05/11/09

This T type external routine should be used whenever a disk directory (folder) entry is required.

CDS185 will:

  1. Include F2=Explore in the prompt to enable browsing to the desired folder.
  2. Convert backslashes \ to slashes /.
  3. Append a trailing slash when required.
  4. Verify that the entry is a valid directory/folder.

Calling Format:

CALL "CDS185", S039$, REC$, X100$

CDS185 Arguments
Argument Passed To/From Description
S039$ To All parameters for entry & display
REC$ To/From The templated string variable containing the S039.FLDNAM$ to be entered
X100$ To/From T type external routine parameters
