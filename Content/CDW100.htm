CDW100

Generate Dynaweb Menu

 Last Revised: 04/25/12

This program can be run from a web browser running Dynaweb to display a menu of options.  It can also be called programmatically and return the menu in a string variable.  These menu's are maintained using Dynaweb Menu Maintenance on WMS menu.  Each menu is presented in an HTML table and may include a) heading rows, b) other Dynaweb programs, c) a hyperlink, and d) a link to a Dynaweb sub-menu.  The menu optionally includes buttons at the bottom to for Home, Back, Close, and Logout.

The url to display a Dynaweb menu is [DSN]?p=CDW100&m=XXX.

Arguments
[DSN] global variable for the Dynaweb Script Name
xxx Three letter menu code as defined in Dynaweb Menu Maintenance (WMS)

Calling Format:

CALL "CDW100", Y$, MENU$, HTML$ {,DD31L$)

Y$ Dynamo Tools Session Control Variable.  The company name Y.COMPNAME$ will be displayed in the top row of the menu.  Set to null to omit this row.
MENU$ Three letter menu code as defined in Dynaweb Menu Maintenance (WMS)
HTML$ String representing the menu in html format, set by CDW100
DD31L$ Optional stack list of DD31$ entries that will be appended to the bottom of the menu.  This option is useful when there are some menu selections that you want to present conditionally.  You can build DD31L$ by starting with an empty DD31$ record, populate the fields as needed, then concatenate onto DD31L$.  DD31L$ can contain one or more DD31$ menu selections.

The menu is styled using a style sheet.  The name of the style sheet is typically specified using Dynaweb Control Maintenance, and is used for all Dynaweb applications.

Menu Section Style Sheet Entry
top row of menu table.menu tr.top td
next row of menu - menu title table.menu tr.title td
menu detail (selections) table.menu td
optional footer row with action buttons table.menu tr.foot td
