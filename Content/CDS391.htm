CDS391

Enter URL (T)

 Last Revised: 07/13/09

This T type called routine can be used to enter/maintain a URL.  T type routines are typically called by the View Manager (CDS550) when used in a File Maintenance application, although application programs can also call T type routines if all arguments are passed.

CDS391 performs the following verification steps if the entry is non-blank:

  1. converts any \ to /
  2. strips leading http:// or HTTP:// from entry if present
  3. appends / if not present

Calling Format:

CALL "CDS391", S039$, REC$, X100$

CDS391 Arguments
Argument Passed To/From Description
S039$ To All parameters for entry & display
REC$ To Data structure containing URL field
X100$ To/From T type external routine parameters
