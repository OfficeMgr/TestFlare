CDS085

Sort String Data File

 Last Revised: 04/09/09

This program is used to sort a string type data file and store the sorted records in another string file.

Records in the file should be terminated with either a line feed, or carriage return and line feed characters.  The entire record is treated as the key beginning in position 1.

This program is often used in a Windows environment where opening a pipe to sort, i.e., OPEN (PIPE)"|sort" is not possible.

Calling format:

CALL "CDS085", FILEIN$, FILEOUT$

CDS085
FILEIN$ Input file name.  Must be string type with line feed or carriage return & line feed separators between records.
FILEOUT$ Output file name.  If the output file name exists, it will be replaced.  If the output file name is null, then FILEOUT$ will be the same as FILEIN$ with an additional suffix of .srt and returned to calling program.
