CDS043

Enter Program Name

 Last Revised: 02/22/08

This X type called routine is used to enter a program name.  The program description is displayed to the right of the program name.  Note that the rest of the display line is cleared, so no other fields should be placed to the right of the entry column.

Refer to text CDX040 for call/enter list used for X type variables.

The Z8$ parameter may be passed with file verification options if desired.

Examples

CALL "CDS043", MinSiz, MaxSiz, Col, Row, "+Program Name+", Z, PROG$, Y, Z8$, FMT[ALL]

CALL "CDS043", 1, 6, col, row, "+Program Name+", Z, PROG$, Y, "{XXCWO4,-}", FMT[ALL]
