CDS047

Enter Operator/Login ID

 Last Revised: 04/26/10

This X type routine can be called whenever an operator ID is to be entered, displayed, or formatted.  The operator ID's maintained with Operator Information Maintenance (SMC) will be valid entries. The operator name can be displayed to the right of the ID.

Calling Format:

CALL "CDS047", Z1, Z2, Z5, Z6, Z0$, Z, Z$, Y, Z8$, FMT$[ALL]

CDS047
Input Variables
Z1 0 - any entry is valid
1 - entry must be in operator file
-1 - blank entry permitted
Z2 not used
Z5 horizontal screen position (col)
Z6 vertical screen position (row)
Z0$ operator message
Z passed as zero; set to length of Z$
Z8$ DIS - display operator ID and name @(z5,z6)
FMT - formatting only, no entry. return operator ID & name in FMT$[1] based on ID specified in Z$
ENT or null - enter operator ID
XY where
X = use prompt window Y/N
Y = display name
R=right of ID
B=below ID
COL,ROW at column specified
Output Variables
Z$ operator ID
FMT$[1] when Z8$="FMT" then
FMT$[1] will contain operator ID (from Z$) followed by a space and the operator name
Y CTL of operator entry
STBL("SMS_CDS069_DIS") name of operator as displayed
