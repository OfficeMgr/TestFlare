CDW279

Convert Templated String to HTML Table & Text

 Last Revised: 02/19/14

This routine takes a string template as input, and produces HTML output in the form of a table. It is frequently used when debugging Dynaweb applications, as you can display in the browser any templated data structure. It also returns the values in text format, suitable for storing in a log file or printing to a display.

CALL "CDW279", REC$, HTML$, { TITLE$, TEXT$, W279$ }

CDW279
REC$ To Templated data structure. Also can be "!LIST" in which case all global variables that do not begin with ! are placed in the table.
HTML$ To/From This program appends a table to the end of the HTML$ passed
TITLE$ To Optional table title will be placed in the first row if passed
TEXT$ To/From Optional. Templated field names and values will be appended to TEXT$ passed with each field on a separate line. Suitable for storing in a log file.
W279$ To Optional. Pass a "T" to show field titles instead of field names when present in the template.
HTML Example

2000 BEGIN

2010 CALL "CDS095",IM27,"DSIM27","",IM27$

2020 READ RECORD(IM27)IM27$

2030 CALL "CDW279",IM27$,HTML$,"DSIM27"

2040 RUN "CDW999"

Note that you can also CALL "CDW999",HTML$ from a program or from console mode when debugging.
DSIM27
Field Name Value
PCL 000
DESC B & D / Dewalt Tool Sales
PCLRENT 500
PCLUSED 797
COST_FACTOR .78
Text Example
2000 BEGIN
2010 CALL "CDS095",IM27,"DSIM27","",IM27$
2020 READ RECORD(IM27)IM27$
2030 CALL "CDW279",IM27$,HTML$,"DSIM27",TEXT$
2040 PRINT TEXT$
DSIM27
PCL=000
DESC=B & D / Dewalt Tool Sales
PCLRENT=500
PCLUSED=797
COST_FACTOR=.78
