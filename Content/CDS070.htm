CDS070

Obtain Number of Records / Bytes Used

 Last Revised: 05/16/99

CDS070 is a called program whose purpose is to determine the number of records used in a file.

Calling Format:

CALL "CDS070", Z, Z2, Z$

CDS070
Z device number of file currently open
Z2 returned as the number of records in use
Z$ name of the file from File Data Dictionary
Example

Determine the number of records in the file DSUD10.

OPEN (1)"DSUD10" CALL "CDS070",1,Z9
