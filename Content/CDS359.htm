CDS359

Format N type numeric

 Last Revised: 11/20/07

This routine should be used to format floating decimal (N type) numeric variables for display or printing.

This routine converts a numeric variable to a string of size specified.  It will include commas when possible and display with a thousands indicator (M) when required.  The last character of OUT$ will contain a "*" character when the numeric value does not fit it the size specified.

Calling Format:

CALL "CDS359", SIZE, VAL, OUT$

Input Data Elements
SIZE The number of characters that OUT$ should contain.  Can be passed with a fractional value such as 9.4 in which case the fractional part is ignored.  Can also be passed as a negative value which causes OUT$ to be all blank instead of containing a zero when VAL is zero.
VAL The value of the numeric to be formatted
Output Data Elements
OUT$ The right justified string representation of VAL.
Examples
>CALL "CDS359",8,123456,OUT$
>?OUT$
 123,456
 

>CALL "CDS359",8,-123456,OUT$
>?OUT$
-123,456
 

>CALL "CDS359",8,123456.5,OUT$
>?OUT$
123456.5
 

>CALL "CDS359",9.4,123456.5,OUT$
>?OUT$
123,456.5
 

>CALL "CDS359",9,-123456.5,OUT$
>?OUT$
-123456.5
 

>call "CDS359",8,123456000,out$
>?out$
123,456M

>CALL "CDS359",8,-123456.5,OUT$
>?OUT$
-123456*

>CALL "CDS359",4,0,OUT$
>?OUT$
   0

>CALL "CDS359",-4,0,OUT$
>?OUT$
