CDS353

Convert text to/from base64

 Last Revised: 06/16/15

CDS353 can be called to convert text to base64 format, or convert base64 text to regular text format.  It is used by CDS184 when emailing attachments, and can be used by application programs that might need to store text in base64 format, as is sometimes done when storing a PDF or graphic file within an xml data structure, or whenever data that might contain binary or non-displayable characters needs to be represented in displayable ASCII characters only.

Calling Format:

CALL "CDS353", TXTIN$, TXTOUT$ {, MODE$ }

Note: CDS353 performs no disc operations, so if the input text is very large and you are not using bbj, you may run out of memory if you try to convert large amounts of data with a single call to CDS353.

CDS353 Arguments
Argument Passed To/From Description
TXTIN$ To Input text
TXTOUT$ From Output text
MODE$ To Indicates the type of conversion
E, null, or not passedConvert TXTIN$ to base64 and return the base64 text in TXTOUT$
DConvert TXTIN$ which is in base64 format, to regular text in TXTOUT$
