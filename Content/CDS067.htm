CDS067

Print File Expansion

 Last Revised: 03/10/99

This program is called by the error routine, CDS063, to expand any full data files. It will also create overflow files when a spool file fills.  This routine is rarely required any more since most file are defined as dynamic and will expand to the limits of the disc drive or file system.

Calling Format:

CALL "CDS067",D,D$

CDS067
D is only passed to CDS067 when you know the device number of the full file. If it is not passed to CDS067, or passed as zero, CDS067 will attempt to determine the device number of the full file.
D$ is information used by the error routine, CDS063, to record the file expansion details on the error log.

Although this routine is usually only called by the error routine, it can be used to expand any full file. If a program does not have sufficient memory, or is not interfaced to the error routine, you may have sufficient memory to expand the file by typing in console mode: CALL "CDS067",X where X is the device number of the full file. Note that CDS067 can only expand files completely full.
