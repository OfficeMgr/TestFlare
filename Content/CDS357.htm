CDS357

Set Parity Bit or Convert to Binary

 Last Revised: 02/13/04

This called routine can be used to convert a string to 7 bit even, odd, or no parity, or to convert a string to a binary representation of ones and zeros.

CALL "CDS357", TXTIN$, PAR$, TXTOUT$

CDS357
TXTIN$ Input string with no length restrictions
PAR$
O Odd Parity
E Even Parity
N No Parity
B Convert to Binary String
TXTOUT$ Output string
Examples
>CALL "CDS357",$0001020304$,"O",TXTOUT$
>?HTA(TXTOUT$)
8001028304


>CALL "CDS357",$0001020304$,"E",TXTOUT$
>?HTA(TXTOUT$)
0081820384

>CALL "CDS357",$0081820384$,"N",TXTOUT$
>?HTA(TXTOUT$)
0001020304

>CALL "CDS357","123","B",TXTOUT$
>?TXTOUT$
001100010011001000110011
