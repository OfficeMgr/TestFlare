CDS018

Compute / Log User Counts

 Last Revised: 01/29/09

This program is used to compute and log user count statistics including operating system users, bbx/bbj users, and dynamo users.  These statistics are written to file CDSM09 and are displayed using User Count Statistics on the SMS menu.  This routine is called by the Dynamo Tools menu processor and is not typically called by application programs.

Calling Format#1 - compute and log all users

CALL "CDS018", Y$, USERS[ALL]

CDS018 - compute and log all users
Y$ session control record
USERS[1] number of active operating system users, Unix systems only.  Determined using 'who | wc -l'.  Note that this may not match exactly the definition of a licensed user for some operating systems.
USERS[2] number of active BBx/bbj users, determined using INFO()
USERS[3] number of active Dynamo users, determined using CDSM08.  Note that the definition of a Dynamo user does not count multiple sessions open on the server console, multiple connections from a single PC running FacetWin, Dynaweb or background tasks, hotkey sessions, nor sessions with the same IP Address or PCNAME if Dynamo Tools can determine these values.

CDS018 also automatically purges daily transaction records that are more than one year old.  It does not purge monthly records.  See Display User Count Statistics on SMS menu for additional information.

 

Calling Format#2 - compute dynamo users for one company

CALL "CDS018::CC", CC$, USERS

CDS018 - compute dynamo users for one company
CC$ two character company code, passed to CDS018
USERS number of active Dynamo users for company code provided
