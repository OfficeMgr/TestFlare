CDS192

Set/Clear Caps Lock/Num Lock

 Last Revised: 09/17/14

This program is used to Set or Clear the Caps Lock and/or Num Lock setting when using Dynastart.

If not using Dynastart, no action nor error will occur.

 

Calling Format:

CALL "CDS192", { CAPSLCK, NUMLCK }

CAPSLCK
0 Set CapsLock Off
1 Set CapsLock On
any other value Do not change CapsLock
NUMLCK
0 Set NumLock Off
1 Set NumLock On
any other value Do not change NumLock
