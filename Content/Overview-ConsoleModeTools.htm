console mode tools

Overview

 Last Revised: 02/10/13

There are several Dynamo Tools that can be called while debugging a program or when investigating an error. These tools are useful to:

  • Display template field names and display/change values
  • Display all files currently open
  • Access the File Utility for an opened file
  • Perform a complete formatted dump
  • Hotkey to another bbx instance, and return
