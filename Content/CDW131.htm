CDW131

Dynaweb File Utility - Display Records

 Last Revised: 02/23/08

This Dynaweb application displays records from a data file using CDW038.  All fields are included.

Arguments
p=CDW131 Program Name
filnam=xxxx File Name
kno=X File KNUM (optional, assumes zero if not passed)
recs=XXX Number of records to display (optional, assumes 500 if not passed)

Example:

Excellware, Inc.
Dynaweb File Utility - Display Records

Product Class, 10 records
