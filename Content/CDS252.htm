CDS252

Return FID and FIN Templates

 Last Revised: 08/26/14

This called routine is used to returns information about an opened file or device.  Best Practice is to use CDS252 and not FID() and FIN() within your application, because CDS252 will be able to support alternate formats of these functions such as XFID() and XFIN().

Note: You can enter FID. and FIN. followed by F2 in the program editor to paste field names.

CALL "CDS252", CHAN, FID$ {, FIN$, S252$ }

Field Usage
.COL$[1] Can be printed to display to restore colors to default values
.COL$[2] Can be printed to display to change foreground color
.PCOL Column for prompt at bottom of active window. Typically 8 for full width windows, and zero for others.
.PROW Row for prompt at bottom of active window, located at 2nd to last row, i.e., 23 for a 25 row window.
