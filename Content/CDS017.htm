CDS017

Remove Inactive Records from Session Control File

 Last Revised: 01/29/09

This routine is called by the Display System Status program and the background Command Processor to remove records from the CDSM08 System Status File when either the process or device name no longer exists.  It is not normally called from application programs.

Calling Format:

CALL "CDS017"

There are no arguments passed to this routine.  The System Status File CDSM08 is opened and closed within CDS017.  When called from a non-background task, an activity message is displayed and cleared.
