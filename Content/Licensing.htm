Licensing

 Last Revised: 08/21/15

Software from Excellware and others who base their application software on the Dynamo Tools is licensed for use based on the number of simultaneous users.

A user is roughly defined as a login session with some exceptions.

  • Multiple sessions from the same personal computer using the FacetWin terminal emulator only count as a single user. Multiple sessions from a personal computer using any other terminal emulator will count as multiple users.
  • Multiple Dynastart sessions from the same workstation count as a single user.
  • Dynamo Tools Dynaweb applications as executed by a web server, counts as a single user, regardless of the number of web processes running.
  • Background sessions, including BBj Stored Procedures and Triggers, do not consume a Dynamo Tools license.

You can purchase additional users at any time by contacting Excellware.  Installation of additional users is done by a brief telephone call to Excellware using the Update License Control Information option on the System Management Configuration (SMC) menu.

The license is transferable to any replacement computer, but not to another company or entity.

New Installations

You will be prompted for a License Control Number.  You can obtain this number from your software provider or Excellware.  You will need to call Excellware to complete the licensing process where you will be given an authorization number to enter.

Replacement Server

All files can be transferred to the replacement server without limitation.

Pre-requisites

Dynamo Tools requires an active license to either Pro5, Visual Pro5, or BBj, all products from Basis International .  Excellware can provide an automated installation process for Basis products that run on a linux distribution that uses 'yum'. Contact Excellware support for additional details.
