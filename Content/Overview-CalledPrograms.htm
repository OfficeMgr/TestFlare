Called Programs

 Last Revised: 02/10/13

Dynamo Tools include a wide variety of programs that can be called from your application.

Using these tools, will reduce your development time, create code that can be more easily maintained, and will produce a more consistent user interface.

You can access the Dynamo Tools Called Program Library from within the Dynamo Tools Program Editor. Simply enter CALL "CD followed by F2.  Then enter some word or phrase such as "Bar Code" or "Telephone" or "Email", then select the routine from the list.

You can also obtain the CALL/ENTER variable list of each Called Program when using the Dynamo Tools Program Editor. Enter the CALL such as CALL "CDS055" then touch F2 to paste the default variable list into your source program.

You can touch F3 while editing a line of code to present a list of Help Topics based on the program statement.
