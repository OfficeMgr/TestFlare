CDW001

HTML Help Set up

 Last Revised: 09/22/04

This program determines if help text is available for a given web application and formats the hypertext link to the help text if present.

The HTML source should include the reference [HELP] where the link is to be inserted. If there is no help text available for a given application the [HELP] will be cut from the HTML source.

CALL "CDW001", ENV$, HTML$, PROGID$

CDW001
ENV$ the BBweb environment variable
HTML$ the HTML page being built. This program will add a small javascript routine on the front of HTML$ which will enable the help window to have no controls.
PROGID$ the web application program ID. It is normally not passed to CDW001, since the PGM(-1) will be assumed.
data info


The help text must reside in a directory named help immediately under the server_root (htdocs) directory of the web server. The help text must have a lower case name and use a .htm suffix.

It is assumed that the [HELP] reference will be contained in a list of links often found in the footer section of the page. A pipe (|) separator will be included before the actual link when help text is present.
