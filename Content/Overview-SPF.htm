Software Packaging Functions

Software Packaging Functions are programs used to bundle together programs and data files to be installed as a package or package upgrade on another system.  

These programs are also applicable only to Unix systems, however, the installation routines are executed upon next user login, so that the same code can be used to install/upgrade Windows based systems.

The software package is installed using Install/Update Software Package on the SMC menu.

Note that the following Package Codes are reserved by Excellware.

SMS Dynamo Tools - Runtime Components
FMS Dynamo Tools - Developer Components
WMS Dynamo Tools - Dynaweb Components
ZIP Dynamo Tools - Zip Code Database
DAP Dynamo Accounts Payable
DGL Dynamo General Ledger
DYN Dynamo - Base Application
DTO Dynamo Transfer Order Processing

This menu of applications is deprecated but still functional, and is still used for Dynamo Tools installation on AIX, Solaris, HP-UX, and other non-linux, non-windows systems.

On linux systems, Dynamo Tools is installed and updated using the yum command. See Dynamo Tools Installation for mode details.
