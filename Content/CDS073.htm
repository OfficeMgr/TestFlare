CDS073

Enter Hexadecimal Data

 Last Revised: 11/06/08

This routine is a program segment that can be used with file maintenance as a custom called variable, type 'C'.

It performs the same as an input routine but places additional restrictions on the entered data. Specifically, the length must be zero or a multiple of 2 characters. The data can contain only valid hex characters, i.e., the digits from 0 to 9, and the letters from A to F.

Examples of valid data

'null string' B1B2B3B4 A0 1B1C

Examples of invalid data

A0A (cannot be an odd number of characters)
AG (cannot contain 'G')
