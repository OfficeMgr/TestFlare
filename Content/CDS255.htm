CDS255

Input Routine - Calendar

 Last Revised: 01/22/08

This is the routine called by CDS069 to display a calendar and prompt the operator to select a date. It can also be used by application programs.

CALL "CDS255",Y$,F4,JUL,COL$

CDS255
Y$ the normal terminal control variable
F4 used to indicate if the F4 key may be used to exit. If F4 is non-zero then the F4 key will exit.
JUL the default date for input, and will contain the selected date at output. If passed as 0, then the current system date will be the default.
COL$ a group of 3 or 4 color numbers. Each color number is 3 characters in length. The first color number indicates the window border, then second indicates the non-selected dates, the third indicates the selected or highlighted date. The 4th color number is optional. If passed it is used for the day of the week headings and prompt line. If not passed then a suitable color is selected by CDS255. If COL$ is passed as null, then the operator's F2 display colors as specified in Operator Preference Maintenance will be used.
