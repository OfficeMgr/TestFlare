CDS400

Enter Printer ID

 Last Revised: 04/07/06

This X type called routine can be used by applications to specify the ID of a defined printer.

If the minimum value is 0, then the entry is optional.

If the Z8$ parameter is specified as LST then the complete list of all defined printers will be returned in a string with 44 characters per printer with the 4 character printer ID followed by the 40 character description.

The printer description will be displayed to the right of the entry.
