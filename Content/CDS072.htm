CDS072

Display / Toggle / Clear Options

 Last Revised: 05/16/99

This public routine is used to display certain options to the operator and display current option.

The format of the display is shown in the example below.

#. HEADING LINE

===============

OPTION 1

OPTION 2

OPTION 3

The routine will support more than the 3 options shown here. The maximum number is limited only by the position on the screen where the options are displayed. The routine will display the current option selected in foreground (brighter screen intensity), and all other options in background (lighter screen intensity).

The first time you call this routine, the routine simply displays the heading line and options on the screen, with the default option in foreground.

Calling format:

CALL J3$=""; CALL "CDS072", J1$, J2$, J3$

where J1$(1,1)

After the initial call, subsequent calls will cause the option to toggle. That means that the current option is cancelled, and the next option is displayed in foreground.

One additional feature is that the routine can clear the data from the screen. This may be needed when some previous operator selection causes this option not to apply.
