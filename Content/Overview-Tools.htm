dynamo tools

Overview

Documentation Published on

Dynamo Tools include programs, files, and procedures that will:

  1. Present a consistent interface to the end-user, regardless of individual developer, reducing end-user training time and operator errors,
  2. Increase programmer productivity,
  3. Produce more readable and maintainable BBx/BBj code,
  4. Produce accurate documentation of data files,

and more!

There are currently four components:

  1. Runtime Components - tools and end-user routines that are installed on each system,
  2. Developer Components - tools used by software developers to describe data files, provide file maintenance capability, and tools used for program writing and debugging.
  3. Dynaweb Components - tools that are used to build BBx/BBj based web applications.
  4. Zip Code Database - data files and programs used for address verification including United States 5 digit Zip Code to City/State lookup.

 

What's New?

Project Deployment

There has been quite a bit of work done lately on what has been called Data Dictionary Export/Import (FMS), including:

  1. Re-titled as Project Deployment and relocated to #9 on FMS menu.
  2. Added support for bbj triggers
  3. Added support for bbj webstart/jnlp applications
  4. Added support for uploading to, and downloading from secure cloud based storage we call Dynastore.
  5. Added support for uploading and downloading using sftp.
  6. Added automated activation process to multiple destination folders based on file type and name, i.e., programs, configuration files, jar files, dynaweb files, images, and scripts,
  7. Completed logic to Maintain (delete) files in the local export and Dynastore folders
  8. Added project activation history
  9. Added project archiving and automated rollback
  10. Added One Step Download/Import/Activation
  11. Added support for strict change control where only selected users can activate projects, and non-privileged users, even the user running bbj services, does not have write permission to the program and other target folders.
