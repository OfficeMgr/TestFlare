CDS182

Return bbj info

 Last Revised: 11/09/09

This program is used by the Dynamo menu I (Information) option and called by CDS030 to return bbj information.

Calling Format:

CALL "CDS182", Y$, TOBBJ$, FROMBBJ$

Input Data Elements
Y$ Session Control Data
TOBBJ$ Dummy placeholder argument as no parameters are needed to be passed to CDS182.
Output Data Elements
FROMBBJ$ Templated data output from CDS182 including BBj Level, Revision, Port#, Build, JVM release, and JVM Path
Example
CALL "CDS030",Y$,"CDS182",10,"",S182$
CALL "CDS279",S182$
