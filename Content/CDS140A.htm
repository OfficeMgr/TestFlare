Program Editor

Statement Editor

 Last Revised: 04/29/09

To edit an existing program statement, enter an apostrophe followed by the line number or line label.   For compatibility with an older version, you can still use the "at sign symbol @" in place of the apostrophe.   When you edit a program statement in this way, the program statement will be displayed and you can edit it using the following commands.

The first column indicates the primary key to press.  The second column indicates the control key to press (Hold Ctrl down while touching key).  The primary key is dependent on the function key mapping established using Terminal Function Key Maintenance (SMC).  The table below assumes use of the STD Function key Group Code.

Cursor Movement
ArrowRt ^L Right 1 character
Tab ^I Right 8 characters
PageDn ^N Right to next semicolon
End ^E One character after last character of statement
F4 ^_
ArrowLf ^V Left 1 character
ShiftTab ^B Left 8 characters
PageUp ^O Left to previous semicolon
Home ^P To first character in statement
ArrowUp ^K Up one line, or same as Home if on first line
ArrowDn ^J Down one line, or same as End if on last line
Inserting
Insert ^Y Toggles between insert (block cursor) and typeover (line cursor) modes
F12 ^Z Inserts blank character
Sh+F12 ^W Inserts 8 blank characters
Sh+Insert ^A Inserts one line (typically 80 characters)
Deleting
Delete ^T Deletes character at cursor
Backspace ^H Insert Mode: Deletes character to left of cursor

Typeover Mode: Blanks character to left of cursor
  ^U Delete all blanks between cursor and next non-blank
Sh+F10 ^G Delete from cursor to end of line
  ^X Delete from cursor through character before semicolon
  ^F Delete from/to.   Touch ^F, move cursor, then touch ^F again.
  ^D Delete one line (typically 80 characters)
Sh+F1 ^R Inserts (restores) last deletion (even if deleted on different program statement or program)
Other Statement Editor Commands
Enter ^M Accept program statement
Escape ^[ Abandon changes made to program statement, restore original
  ^C Abandon changes made, exit statement editor
F1 ^\ Hotkey
F2 ^[ Paste line drawing character or context sensitive code.  See Lookup and Paste Commands below.
F3 ^^ Help - display Statement Editor Commands
