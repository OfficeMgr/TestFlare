CDS273

Remove Records

 Last Revised: 02/23/14

This routine removes records from keyed files. It can be used to remove a single record, or multiple records with the same starting key characters.

CALL "CDS273",ERR=NNNN,Y$,FILNAM$,KNO,KEYVAL$ {,RECS,S273$ }

CDS273
ERR=NNNN Branch to this line or line label if the file cannot be opened, or if KEYVAL$ exceeds the key length for the KNO.
Y$ Session control variable.
FILNAM$ Name of the file where records are to be removed.  XX is permitted as the company code.
KNO For MKx files, the KNUM of the file corresponding to the KEYVAL$ provided.
KEYVAL$ Starting characters of the key(s) to be removed.
RECS Optional, and if present CDS273 will return the number of records removed.
S273$ Optional, pass as "N" to bypass user prompt if unable to open FILNAM$.

CDS273 calls CDS036 to remove the record so that it will be logged for synchronization with another server if required.
