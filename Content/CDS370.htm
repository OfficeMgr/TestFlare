CDS370

Purge Fax & Email Logs

 Last Revised: 10/30/15

This application can be run from the PFM menu, but can also be called or run by a background task to periodically purge the Dynamo Tools Fax and Email log files.

Calling Format:

CALL "CDS370", DAYS

Typical cron task entry which purges the log files at 3:00 AM on the first of each month. The 60 argument represents DAYS.

0 3 1 * * /u/CDI/bin/background CDS370 CD Purge_Dynamo_Tools_Log_Files 60

Input Data Elements
DAYS Purge transactions older that this number of DAYS.

This program will remove records from the Dynamo Tools Log files listed below. Note that file CDSM96, the Outgoing Email Summary Log file is not purged as it typically only contains one record per day.

If running on a Unix/Linux/AIX server, and vsifax is installed, then this program will also purge the vsifax log files. Note that this option requires that the Dynamo Tools Command Processor be running, unless run by 'root', as would be the case if root's crontab was used to schedule this task.

File Name Description
CDSM90 Fax Modem Log (vsifax)
CDSM91 Fax Modem Attempt History (vsifax)
CDSM92 Dynafax Log
CDSM95 Email Log
