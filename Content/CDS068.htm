CDS068

Input Routine Subscript Numbers (OBSOLETE)

 Last Revised: 12/01/82

CDS068 is a small called routine to simplify the normal calling sequence to CDS069 when using a numeric array. For example: the following calling sequence to CDS069:

NOTE

This program is obsolete and is no longer provided. Upgrades from older MAI Basic Four installations require replacement of all calls to CDS068 with a call to CDS069.

Z=D(10); CALL "CDS069", 0, 9999, 67, L,"+COST", "#,###.00", Z, ""; D(10)=Z

can be replaced with a more simple sequence:

CALL "CDS068", 0, 9999, 67, L, "+COST", "#, ###.00", D(ALL), 10

The variables are the same for the two routines up to the numeric mask. After that CDS068 requires some numeric array. All elements must be sent to the routine using the ALL option. The next and last variable required is a numeric constant or expression indicating which element of the array is to be used.

CDS068 will save PSZ over CDS069 and in addition changing D(10) to D(11) will only require a single change to the calling sequence as opposed to two which would be required if CDS069 was used.
