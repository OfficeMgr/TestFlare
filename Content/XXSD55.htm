File Maintenance Record Layout

Purpose: Generates VDT's in XXSD53 and lengths of all fields in File Maintenance.

Iolist: F0, N, M$, D0$, D2$, D3$, D$, F$, D1$, D5$, D6$, D7$

XXSD55
F0 number of Data Elements in the key
N number of Data Elements selected to be maintained (how many are displayed)
M$ string of all masks that are needed
D0$ string of all descriptions, plus, minimum and maximum lengths of each Data Element
D2$ string of all Valid Strings
D3$ string of lengths of each field (3 bytes per field)
D$ all information you need for each Data Element
(27 Bytes per Data Element)
(1,1) field type

L

Left

A

alphanumeric

R

right

N

numberic

(2,1)length of maximum
(3,4)X,Y position of the input (display) on the screen
(7,1)number of fractional positions
(8,1) Maintenance Code (0-7 possible)

0

open to anything

1

must be zero to delete

2

no modifications allowed

3

no modifications allowed zero to delete

4

no adds of new records

5

no to add and zero to delete

6

no to add or modify

7

no add, modify and zero to delete

(9,1)unused
(10,1)field number
(11,1)length of substring
(if part of a variable the number is how long that part is)
(12,3)starting byte within the field
(15,3) for numeric its the starting position of the Mask in M$,

for alpha-numeric its the starting position of the Valid String in D2$

(18,1)length of Mask or length of Valid String
(19,3)used by F/M its the starting position of this Data Element in the record
(22,3)starting byte within D0$ of the description (minimum and maximum)
(25,1)length of description
(26,1)length of the minimum
(27,1)actual length of this Data Element within the record
(also adjusts when maintained)
F$(1,30) names of auxiliary files
(31,1)audit (Y/N)
(32,1)delete (Y/N)
(33,30)deletion files
D1$ 3 bytes for each variable that is a required key
(1,1)what Data Element is in question
(2,1)what file is it a required key in
(3,1)what field to display
D5$
(1,1) length of all Sort Keys
(2,X) where X is the number of data elements that are part of auxiliary sort keys, times 3
(3,4,5,6,7 shows as auxiliary 1-5 in FDD)

A.

what file number

B.

starting byte

C.

length
D6$ fields that must be zero to delete
D7$ comparable to D1$ 3 bytes for each Data Element that is part of a Sort Key
(1,1)what Data Element in question
(2,1)starting bytes in the Sort Key field
(3,1)length of the Data Element
