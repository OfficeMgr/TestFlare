CDS086

Obtain Field Lengths

 Last Revised: 05/16/99

This called routine is used to obtain the lengths of all fields for a record that file maintenance has been generated for. Note that file maintenance must be generated for the desired file and suffix.

Calling Format:

CALL "CDS086", F$, F1$, FLD$[ALL]

CDS086
F$ you pass to the routine F$ as the company code, the last four characters of the file name plus the suffix code indicating the specific record in the file desired.
F1$ the routine returns back F1$ as a string of composed of three bytes for each field in the file. The length of field 1, would be the NUM(F1$(1,3)), field 2 would be NUM(F1$(4,3)), etc. If a field has a variable length, then F1$ will contain zero for that field.
FLD$[ALL] would be dimensioned to the number of fields in the file. The array element for each field that is a fixed length is filled to the proper length for that field. This array may be used to read the file contained in F$ after it is returned to the calling program. The first field is FLD$[1].
ERR branch is taken if a) the file maintenance file XXSD55 is not found for the company specified. A File missing message will be displayed. b) the record requested is not on file. No error message is displayed. The ERR system variable will be 11.
Example

CALL "CDS086", ERR=EOJ, Y.COMPCODE$+"CM01A", F1$, FLD$[ALL]

F1$ will contain field information for the customer master file Y.COMPCODE$+"CM01" suffix A.

FLD$[ALL] will be dimensioned to the number of fields in file Y.COMPCODE$+"CM01".
