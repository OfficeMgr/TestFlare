CDS066

Print Terminal Image

 Last Revised: 04/18/15

This program can be used to print the current contents of the display to a printer.

CDS066 will print standard alphanumeric characters as well as the line drawing characters used for windows borders. It will also print any text underlined on the display, as underlined on the printer.

Note that the entire display is printed, including any visible windows and their borders. Both 80 column and 132 column display widths are supported.

Calling Format:

CALL "CDS066",CHAN,COPIES,Y6$

CDS066
CHAN The device number to use for the printer. CDS066 will use an available channel if not passed or passed as zero.
COPIES The number of copies to be printed. Set to 1 if not passed.
Y6$ The standard printer control variable provided by CDS084. If you pass Y6$ to CDS066, then you have previously opened the printer. In this case, the message that indicates the operator ID, date and time printed does not appear on the printed page. In addition trailing blank lines are not printed. Y6.L is decremented to indicate the number of lines left to be printed on the page.  The printer channel is not closed in this case so that the calling application can print additional data to the same channel.

There is another option to include documentation notes below the printed display image. This option is enabled by setting the global variable SMS_CDS066.PAR. This global consists of two Y or N characters. The first determines if the date and time message should print. The second determines if the documentation note should print. If set to Y, then the operator will be prompted for the text ID, which is stored in the XXSK57 Help Text File. The operator can maintain the text if desired, then it will be printed below the display image.
