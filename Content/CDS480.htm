CDS480

Time Entry HH:MM AM format (T Type)

 Last Revised: 03/03/09

This T type called routine can be used for time entries that are stored in a numeric variable in the bbx TIM format, i.e., hours and fractions of an hour.  CDS480 is typically specified as the program name to call for time entries in the Data Dictionary.

The routine is flexible and allows many different entries such as:
115
115p
115P
01:15
01:15 PM

The AM/PM designator will be added if not entered, based on a typical 8am to 5pm work day.

Calling Format:

CALL "CDS480", S039$, REC$, X100$

CDS480
S039$ Templated parameters for entry
REC$ The data structure containing the numeric time field, i.e., NFIELD(REC$,S039.FLDNAM$,S039.INDEX)
X100$ Standard T type routine parameters including X100.ACTION$
Example
1010 BEGIN
1020 PRINT 'CS',
1030 CALL "CDS276","Start Time",COL
1040 DIM REC$:"START:B"
1050 CALL "CDS041","CDS039",S039$,"YY"
1060 LET S039.FLDNAM$="START",S039.INDEX=1,S039.COL=COL,S039.ROW=2
1070 CALL "CDS041","CDX100",X100$,"YY"
1080 CALL "CDS480",S039$,REC$,X100$
1090 CALL "CDS279",REC$
