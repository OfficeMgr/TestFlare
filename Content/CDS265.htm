CDS265

Get Month and Week Names

 Last Revised: 02/26/09

This program will return a templated string variable containing:

  • Default date format mask
  • Month names in short and long format (Jan & January)
  • Day names in short and long format (Sun & Sunday)

Calling format:

CALL "CDS265",S265$

CDS265
FORMAT default mask for the DATE() function %Mz/%Dz/%Yz
MTHS[12] month names - short Jan - Dec
MTHL[12] month names - long January - December
DAYS[7] days of the week - short Sun - Sat
DAYS[7] days of the week - long Sunday - Saturday


This data is retrieved from the default global string !DATE.  Using CDS265 instead of hard coded month and day names is preferred, since changing the !DATE global and using CDS265 provides the ability to change the language, i.e., English vs, German, etc. without changing the application.
