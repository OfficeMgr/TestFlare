CDS035

Return File Restriction Indicator

 Last Revised: 08/04/95

This called routine is used in conjunction with Inter-System Communication to determine if a particular file is transmitted to this system, and therefore controlled by another system.

Calling Format:

CALL "CDS035", Y$, FILNAM$, STS, IS00$

CDS035
Y$ the normal task control variable
FILNAM$ the file name to be checked. If the file name is only four characters long, or if the first two characters are XX, FILNAM$ will be modified to include the company code
STS
0 FILNAM$ is controlled by another system and the file is sent to your system.  Applications should not modify this file since its contents may be overwritten by the controlling system.
1 FILNAM$ is controlled locally, and writes and removes are permitted.
IS00$ optional, and will include the XXIS00 record for the requested file if present

If the Inter-System files to log control file XXSI00 is missing, or does not contain a record for FILNAM$, or the file name is not 6 characters long, then STS will be returned as 1.
