SW110

Contact Info Lookup Routine

 Last Revised: 07/12/11

SW110 is the default routine called during Printer Selection and Memo processing to obtain customer, vendor, salesman, or other contact information.  You can specify your own routine to use instead of SW110 using Company Info Maintenance (SMC).  Use of a Contact Info Lookup Routine can eliminate manually entering contact information including contact name, company name, phone#, fax#, email address etc.

Calling Format:

CALL "SW110", Y$, SW110$

SW110 Arguments
Field Passed To/From Type Description
Y$ To templated Session Control Variable
SW110.NAME$ From C(30) Contact name.  The Contact Info Lookup Routine should set this field to the Contact Name.
SW110.COMP$ From C(40) Company Name
SW110.PHONE$[4] From C(21) Up to 4 telephone numbers
SW110.EXT$[4] From C(5) Telephone Extension related to each telephone number above when applicable
SW110.TYPE$[4] From C(1) Telephone Number Types.  See CDS468 for list of valid telephone type codes.
SW110.EMAIL$ From C(40) Email Address
SW110.REQFAX To N(1) Passed as 1 when calling application is performing a fax operation, so a fax number (type=F) is required.
SW110.REQMODEM To N(1) Passed as 1 when calling application is performing a modem operation, so a modem telephone number (type=M) is required.
SW110.REQEMAIL To N(1) Passed as 1 when calling application is performing an email operation, so an email address is required.
SW110.ENTRY$ To C(40) Passed to Contact Info Lookup Routine of search text entered by calling program.
