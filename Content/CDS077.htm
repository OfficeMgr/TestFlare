CDS077

Obtain Program Names from Menu

 Last Revised: 12/02/04

This called program will select program names from a specified company's menu file. The routine is used in when creating filelists.

Format Options:

CDS077
XX+SEL All programs & links in company XX (XX & DI for example)
XX-SEL Only programs & links in company XX that begin with XX or ZZ
XXcSEL Only programs & links in company XX that begin with CD
XXdSEL Only programs & links in company XX that begin with DI
XXmSEL Only file maintenance data file names
