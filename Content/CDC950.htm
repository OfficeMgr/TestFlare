cdc950

ISC - Prepare Files for Transmission

 Last Revised: 05/20/01

This program is used in conjunction with CDS036 to prepare a file for transmission to another system.

CDS036 logs records written and removed by writing records to file XXIS01. CDC950 reads XXIS01 and writes serial files named ISC/XXIS10.SSS where SSS is the system number specified in ISC System Information Maintenance. The XXIS10.SSS files are the ones that are then transmitted to the remote or headquarters system, although the file name is changed to ISC/XXIS20.SSS on the receiving system. This is done to identify files to be transmitted as XXIS10 and files received to be XXIS20.

CDC950 can be run or called with no arguments. It does not display any progress data so that it can be run in the background. CDC950 removes records from XXIS01 as they are processed.
