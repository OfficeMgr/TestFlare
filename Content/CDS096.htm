CDS096

Setup Terminal and Soft Load Keys

 Last Revised: 05/22/04

This program is called to set the terminal characteristics and function and editing key assignments.

The terminal characteristics are determined based on the terminal type and the operator preference items specified in Operator Preference Maintenance. If no Operator Preference record exists for the operator currently logged in, certain default conditions are assumed.

Calling Format:

CALL "CDS096", Y$, U0, U0$

CDS096
Y$ the terminal control variable
U0 the channel number of file SYSD02 if opened by calling program. If SYSD02 is not open by the calling program, pass a 0 to CDS096 instead of the variable U0.
U0$(1,4) the function key group code to be used. If blank, then the default function key group code contained in the operator preference record is used.
U0$(5,1) if blank, then the terminal setup options such as cursor style, status line, reverse screen options, etc. are sent to the terminal as well as the function and editing key values. If 'F' then only the function key values are sent, no terminal setup information is sent.

Note: If U0$ is null, then it is equivalent to all blanks.

Example

CALL "CDS096", Y$, 0, ""

This example sets up the terminal and sets all function and editing keys on the terminal. CDS096 must open SYSD02 locally.

CALL "CDS096", Y$, 10, "W3.2F"

This example will not setup the terminal, but only set function and editing keys associated with the W3.2 function key group code. CDS096 will use channel 10 for all accesses to SYSD02 which was previously opened by the calling routine.
