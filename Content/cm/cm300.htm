Inquiry Functions

Salesperson Inquiry 

 Last Revised: 12/15/14

Salesperson Inquiry will display all information on file for each salesperson.

Upon entering Salesperson Inquiry, the above screen is displayed. This is where a salesperson can be selected. If known, the salesperson number may be typed in. For a list of all available salespersons, press the F2 key.

 

After selecting a salesperson, any of the options displayed in the above image can be selected.

 

The Sales & Charts option will look somewhat like the above image. It will display all product category and product class sales for each year he or she was a salesperson at the company.

 

Selecting "Orders" will display all sales orders that were taken by the current salesperson. The view displays the date, status, invoice number (when applicable), the branch, the warehouse, and the PO#/Reference of the sales order. It also displays the value of the order.

 

Selecting "Invoices" displays all invoices related to this salesperson. This view displays the sales order information related to the invoice, including the order value.

 

Selecting "Quotes" will display all quotes that were taken by the current salesperson. This view will display the date, the quote number, the status (which will just display as Quote), the branch, the warehouse, the PO#/reference, and the order value of the quotes.
