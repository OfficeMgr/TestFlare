CDS250

Display Open File Information

 Last Revised: 03/09/13

The purpose of this routine is to display information about one or all files/devices currently open.  It is typically called from console mode.  The user can highlight and select an open channel to invoke the File Utility on that file.

Calling Format:

CALL "CDS250" {, CHAN, OUTCHAN }

CDS250
CHAN If CHAN is passed, then the file information is displayed for the file opened to CHAN only
OUTCHAN If OUTCHAN is passed, then the file information will be printed to the output device previously opened to OUTCHAN

When OUTCHAN is not passed, or passed as 0, then CDS250 will display the output in a full screen window, and pop it upon exit.
Example
CALL "CDS250"
