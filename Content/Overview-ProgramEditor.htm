Program editor

Overview

 Last Revised: 06/11/14

The program editor is a tool used to write and maintain tokenized BBx programs and tokenized BBj programs that have line numbers.  It includes capabilities to:

  • Enter new program statements
  • Execute BBx commands
  • Edit existing program statements
  • Create & Maintain dated program revision notes
  • Logs date of last save
  • Combine one program statement with another
  • List program statements in formatted (indented based on syntax) or un-formatted
  • Add the program editor interface
  • Add the standard error routine interface
  • Search for text
  • Search for text and edit statement
  • Search for invalid template field names and edit
  • Search and replace
  • Merge program statements from another program
  • Code Completion/Paste Functions including
    • graphics characters (line drawing)
    • line labels
    • template field names
    • Called routine libraries
    • Called routine argument list
    • Defined functions & argument list
  • Import a VDT format
  • Direct interface to display documentation on all Dynamo Tools called routines using web-browser from FacetWin Terminal Emulation Client, bbj thin client, Power Term terminal emulator, or from Visual Pro5 Applications.
