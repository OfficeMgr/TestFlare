CDS050

Enter Company Code

 Last Revised: 02/12/09

This X type called routine can be used to prompt operator for a Dynamo company code, or to return a list of company codes and company names.

Dynamo company codes are two non-blank characters.  Both characters cannot be numeric.  This routine looks for the XXDD00 Company Information file in the XX/ directory, where XX is the company code.

The operator will be able to touch F2 to select from a list of companies.  The list is sorted by company name, not company code.  Excellware company codes CD, DI, DS, and DY will be displayed at the bottom of the list.

Calling Format:

CALL "CDS050", MINLEN, MAXLEN, COL, ROW, PROMPT$, CCLEN, CC$, { CTLVAL, OPT$, FMT$[ALL] }

CDS050
MINLEN To Pass as zero to make the entry optional, otherwise pass as 2
MAXLEN To Pass as 2
COL To Entry column
ROW To Entry row
PROMPT$ To Prompt used for entry.  If passed as null, then the operator prompt will be "Enter Company Code, F4=End".
CCLEN From Will be returned as the length of CC$.  Zero if no entry was made, 2 is an entry was made, or longer when retrieving the list of all company codes on the system.
CC$ To/From Default company code, and company code entered.  List of company codes followed by 40 character company name when requesting a list of companies.  If passed as null or blanks, then the current company code will be the default.
CTLVAL From Set to the CTL value for the entry.  This is sometimes needed when using the ArrowUp or Down keys in a view.
OPT$ To
null Regular Entry
"DIS" Display company code @(COL,ROW) with company name to right
"FWM" same as regular entry but force window mode where user does not need to touch F2 to display list of companies on system
"FMT" return company code followed by blank and name in FMT$[1]
"LST" return list of all company codes followed by 40 character company name in CC$
