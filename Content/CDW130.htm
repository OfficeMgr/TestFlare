Dynaweb File Utility

 Last Revised: 02/23/08

The Dynaweb File Utility is a display only version of the Dynamo File Utility.

You can optionally pass a file name argument in the url.

Arguments
p=CDW130 specifies program name
filnam=xxx optional parameter to bypass entry page, and display information on file name xxx.

Example:

Excellware, Inc.
