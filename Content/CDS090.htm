CDS090

Add / Delete File to Menu

 Last Revised: 02/22/99

 This program is not normally called by application programs. It is used to add or delete a file name to a menu for use in the "F" for files option.

Calling Format:

CALL "CDS090", S0$, S9$, F9$, S$, U

CDS090
S0$ menu code
S9$ list of all menu codes
F9$ four character file name to be added/deleted
S$ function code:
 
AAdd to list
DDelete from list
 U Device number of XXSI50 menu file opened by calling program
